# Battle UI Spec

## 设计基准

目标分辨率：1920x1080

所有坐标均基于 1920x1080。
实际窗口缩放时使用：

scale = min(windowWidth / 1920, windowHeight / 1080)

所有 UI 组件按 scale 等比缩放，并整体居中。

## 资源映射

资源目录：

src/main/resources/assets/ui

已有资源大致用途：

- 战斗背景.png：战斗背景，铺满全屏。
- 狂战士立绘.png：玩家角色立绘，放在左中区域。
- 能量球.png：左下角能量显示。
- 抽牌堆.png：左下角抽牌堆。
- 弃牌堆.png：右下角弃牌堆。
- 结束回合按钮.png：右下方结束回合按钮。
- 设置按钮.png：右上角设置按钮。
- 牌组按钮.png：右上角牌组按钮。
- 金币.png：顶部金币显示。
- 狂战士头像.png：左上角玩家头像。
- 普通卡.png、稀有卡.png、传说卡.png、特殊卡.png：卡牌模板。
- 战斗场景预览.png：旧预览，仅参考，不作为最终目标。

## 整体布局

### 背景

- 使用 战斗背景.png
- 位置：x=0, y=0
- 尺寸：1920x1080
- 铺满窗口
- 不要裁掉中心城堡和月蚀区域

### 左上角玩家状态栏

位置：

- x=20
- y=20
- width=380
- height=115

内容：

- 左侧头像，约 100x100
- 右侧角色名
- 下方生命值
- 整体风格为暗黑金属边框
- 不使用普通矩形背景

### 顶部资源栏

位置：

- x=410
- y=30

内容：

- 金币图标 + 数字
- 其他资源图标可以暂时隐藏，除非已有对应资源

### 右上角按钮区

位置：

- y=20
- 从右往左排列

按钮：

- 设置按钮
- 牌组按钮
- 可以预留 3 个按钮槽位

每个按钮尺寸：

- 55x55 左右

### 玩家角色

位置：

- x=430
- y=330
- height=390 左右
- 面向右侧

注意：

- 不要把角色放得太小。
- 不要放在屏幕正中心。
- 角色脚底应接近地面区域。

### 敌人角色

位置：

- x=1250
- y=300
- height=470 左右
- 面向左侧

如果没有敌人立绘资源，临时使用现有敌人图片或代码已有敌人图，但不要使用默认图形。

### 玩家血条

位置：

- x=420
- y=235
- width=260
- height=45

样式：

- 显示角色名
- 红色生命条
- 数字格式：72/72
- 必须贴近玩家头顶区域

### 敌人血条

位置：

- x=1240
- y=210
- width=300
- height=50

样式：

- 显示敌人名
- 红色生命条
- 数字格式：120/120
- 必须贴近敌人头顶区域

### 左下角能量球

位置：

- x=45
- y=750
- width=145
- height=145

文本：

- 中央显示 3/3
- 下方显示 能量
- 字体大，颜色偏紫白，有阴影

### 抽牌堆

位置：

- x=70
- y=910
- width=110
- height=120

文本：

- 数字显示在卡堆附近
- 不要压住卡堆主体

### 手牌区域

位置：

- 中心点：x=960, y=900
- 卡牌数量：5
- 单卡尺寸：约 190x280

扇形排列：

第 1 张：
- x=560
- y=810
- rotate=-12

第 2 张：
- x=760
- y=790
- rotate=-6

第 3 张：
- x=960
- y=780
- rotate=0

第 4 张：
- x=1160
- y=790
- rotate=6

第 5 张：
- x=1360
- y=810
- rotate=12

要求：

- 卡牌底部可以略微超出屏幕下边缘。
- 卡牌之间可以有部分重叠。
- 鼠标悬停时卡牌上移并放大。
- 不要使用普通矩形卡牌。
- 使用 assets/ui 中的卡牌模板 PNG。

### 右下角结束回合按钮

位置：

- x=1580
- y=760
- width=280
- height=90

要求：

- 使用 结束回合按钮.png
- 文字叠加在按钮中央
- 不要用 JavaFX 默认按钮背景

### 右下角弃牌堆和消耗堆

弃牌堆：

- x=1550
- y=910
- width=110
- height=120

消耗堆：

- x=1710
- y=910
- width=110
- height=120

显示数字：

- 数字显示在卡堆上方或中间偏下
- 字体与整体 UI 匹配

## 禁止项

- 禁止使用第二张图中的大黑色信息面板。
- 禁止用白色默认 ProgressBar。
- 禁止用普通灰色按钮。
- 禁止让卡牌正正方方排成一排。
- 禁止把玩家和敌人做得太小。
- 禁止把血条放在屏幕上方大面板中。
- 禁止为了布局方便而改变核心战斗逻辑。
